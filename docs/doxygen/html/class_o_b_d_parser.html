Detailed Description

Parses OBD-II telemetry data from a CSV file.

Expected columns (0-indexed): 2=Timestampms, 5=Speed, 7=RPM, 8=AbsLoad, 9=OAT, 10=ShortTermFuelTrim, 17=log_MAF.
