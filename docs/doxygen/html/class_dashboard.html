Detailed Description

Renders a real-time instrument cluster (dashboard) onto an OpenCV frame.

The dashboard occupies the left 640×480 region of the combined canvas and displays speedometer, tachometer, temperature, fuel level, and throttle gauges along with a colour-coded driving-style label.
