Detailed Description

Renders the Driver Monitoring System (DMS) overlay onto the shared canvas.

Occupies the right 640×480 region of a 1280×480 canvas. Draws the raw camera feed, face corner markers, eye/head status indicators, and coloured alert banners when drowsiness or distraction is detected.
